# Omsipath

This is a tool for mod developers of the game [_OMSI 2_](https://store.steampowered.com/app/252530/OMSI_2_Steam_Edition/) by [Aerosoft](https://www.aerosoft.com/de/).

When a mod for OMSI2 is packaged, developers typically provide an archive with a folder structure that can be extracted directly into the main directory. This tool allows you to create such structures with just a few clicks and drag&drop. You can also create backups for individual files.

The application was developed using [Microsoft Visual Studio 2022](https://visualstudio.microsoft.com/de/) and requires the [.NET 6 Runtime](https://dotnet.microsoft.com/en-us/download/dotnet/6.0/runtime).
